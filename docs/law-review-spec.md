### 1. 대지 안의 피난 및 소화 통로

- **적용 위치**: 대지 (건축물 외부)
- **정확한 법령명**: 건축법 시행령
- **조문**: 제41조 (대지 안의 피난 및 소화에 필요한 통로 설치)
- **별표**: 없음
- **적용 대상**: 연면적 합계 2,000㎡ 이상 건축물 (모든 용도)
- **기준 내용**:
  - 건축물 주위에 너비 **3m 이상** 소화·피난 통로 확보
  - 주요구조부 내화구조 + 불연재 마감 시 **1.5m**로 완화 가능
  - 건축물 각 부분에서 도로 또는 공지까지 통로 연결
- **필요 입력값**: 연면적, 주요구조부 내화 여부
- **판단 가능 여부**: 연면적 입력 시 해당 여부 판단 가능. 통로 실제 폭·경로는 도면 없이 판단 불가
- **공식 URL**: https://www.law.go.kr/법령/건축법시행령/제41조
- **confidence**: confirmed
- **기존 시스템 매칭**: `category:"카"`, `항목:"대지안의 피난통로"` — 법령이 **제90조의2로 오기**, 내용이 "막힌 복도" 기준으로 잘못 채워짐
- **조치**: 제41조 내용으로 전면 교체. "막힌 복도"는 분리 신설

**현행 코드 (오류):**
```typescript
items.push({ category:"카", 항목:"대지안의 피난통로", 법령:"건축법 시행령 제90조의2",
    내용:"막힌 복도: 너비 **1.2m↑**, 길이 **10m↓**",
    해당여부: Y("막힌 복도 너비 1.2m↑, 길이 10m↓ 확보") });
```

**수정 후 (목표):**
```typescript
const 통로해당 = hasTotalArea && 면 >= 2000;
items.push({ category:"카", 항목:"대지안의 피난·소화 통로", 법령:"건축법 시행령 제41조",
    내용:"연면적 **2,000㎡↑**: 건축물 주위 너비 **3m 이상** 소화·피난 통로 확보 (내화구조+불연재 시 1.5m 완화)",
    해당여부: !hasTotalArea ? W("판단불가 — 연면적 확인 필요")
              : 통로해당 ? Y("대지 내 소화·피난 통로 3m↑ 확보 필요 (도면 확인 요)")
              : N,
    설계기준: 통로해당 ? "건축물 각 부분 → 도로/공지까지 연속 통로. 소방차 접근 동선 확보" : null });
```

### 2. 막힌 복도 기준

- **적용 위치**: 건축물 내부
- **정확한 법령명**: 주택건설기준 등에 관한 규정 / 건축물의 피난·방화구조 등의 기준에 관한 규칙
- **조문**: 주택건설기준 등에 관한 규정 **제16조** (복도), 피난·방화규칙 **제15조의2**
- **별표**: 없음
- **적용 대상**: 복도형 공동주택, 다중이용시설
- **기준 내용**:
  - 막힌 복도(dead-end corridor): 직통계단에서 막힌 끝까지 **20m 이하** (공동주택 기준)
  - 일반 건축물: 직통계단까지의 보행거리 기준으로 간접 규제 (건축법 시행령 제34조)
- **필요 입력값**: 용도, 복도 형식(복도형/계단실형), 층수
- **판단 가능 여부**: 용도·층수로 해당 여부 표시 가능. 실제 길이는 도면 필요
- **공식 URL**: https://www.law.go.kr/법령/주택건설기준등에관한규정/제16조
- **confidence**: estimated — 막힌 복도 20m 기준은 주택건설기준이 출처. 건축법 시행령 직접 조문 아님
- **기존 시스템 매칭**: `category:"카"` "대지안의 피난통로" 내에 잘못 포함 중
- **조치**: **별도 항목으로 신설** (`category:"다"`, `항목:"막힌 복도"`)

**신설 코드 (목표):**
```typescript
const 복도형 = 공동주택; // 복도형 주택 여부 (현재 입력값으로 판단)
if (복도형) {
  items.push({ category:"다", 항목:"막힌 복도 길이", 법령:"주택건설기준 등에 관한 규정 제16조",
    내용:"복도형 공동주택 막힌 복도(dead-end): 직통계단으로부터 **20m 이하**",
    해당여부: Y("막힌 복도 20m↓ 확보 필요 (도면 확인 요)"),
    설계기준:"막힌 복도 끝 → 가장 가까운 직통계단까지 20m 이하",
    confidence: "confirmed" as Confidence });
}
```

### 3. 일반 복도 너비

- **적용 위치**: 건축물 내부
- **정확한 법령명**: 건축법 시행령
- **조문**: 제48조 제1항 (계단·복도 및 출입구의 설치)
- **별표**: 건축물의 피난·방화구조 등의 기준에 관한 규칙 제15조
- **적용 대상**: 거실이 있는 모든 건축물 복도
- **기준 내용**:

  | 복도 형식 | 일반 | 의료·노유자시설 | 초·중·고교 |
  |---------|------|--------------|---------|
  | 양측 거실 | 1.8m↑ | 2.4m↑ | 2.4m↑ |
  | 한쪽 거실 | 1.2m↑ | 1.8m↑ | 1.8m↑ |

- **필요 입력값**: 용도, 복도 형식
- **판단 가능 여부**: 용도 입력 시 해당 기준 표시 가능. 실제 너비는 도면 필요
- **공식 URL**: https://www.law.go.kr/법령/건축법시행령/제48조
- **confidence**: confirmed
- **기존 시스템 매칭**: `category:"타"`, `항목:"복도너비"` — 내용 정확하나 **학교 기준(2.4m) 누락**
- **조치**: 학교 용도 조건 추가

**수정 후 (목표):**
```typescript
const 의료노유자 = ["의료","노유자","요양","장애인"].some(u => 용도.includes(u));
const 학교용도 = 용도.includes("학교") || 용도.includes("교육연구");
const 복도기준 = (의료노유자 || 학교용도)
  ? "양측거실 2.4m · 한측거실 1.8m"
  : "양측거실 1.8m · 한측거실 1.2m";
items.push({ category:"타", 항목:"복도 너비", 법령:"건축법 시행령 제48조",
    내용:"양측 거실: **1.8m↑**, 한쪽 거실: **1.2m↑**. 의료·노유자·학교: 각 **2.4m / 1.8m**",
    해당여부: Y(`${복도기준} 이상 확보`) });
```

### 4. 계단 치수

- **적용 위치**: 건축물 내부
- **정확한 법령명**: 건축법 시행령
- **조문**: 제48조의2 (계단의 설치기준)
- **별표**: 건축법 시행령 **별표1의2** (계단 설치기준)
- **적용 대상**: 모든 계단 (직통·피난·특별피난 포함)
- **기준 내용**:

  | 구분 | 단높이(챌면) | 단너비(디딤면) | 계단 유효너비 |
  |-----|-----------|------------|------------|
  | 일반 건축물 | 20㎝ 이하 | 24㎝ 이상 | 120㎝ 이상 |
  | 초·중·고교·문화집회·위락·판매 등 | 18㎝ 이하 | 26㎝ 이상 | 150㎝ 이상 |
  | 주택 내부 (세대 내) | 20㎝ 이하 | 20㎝ 이상 | 90㎝ 이상 (완화) |
  | 공동주택 공용 | 20㎝ 이하 | 24㎝ 이상 | 120㎝ 이상 |

- **필요 입력값**: 용도, 층수, 연면적
- **판단 가능 여부**: 용도별 기준 표시 가능. 실제 치수는 도면 필요
- **공식 URL**: https://www.law.go.kr/법령/건축법시행령/제48조의2
- **confidence**: confirmed
- **기존 시스템 매칭**: **누락** — judge.ts에 계단 치수 항목 없음
- **조치**: **신설 필요**

**신설 코드 (목표):**
```typescript
const 계단강화용도 = ["학교","문화집회","위락","판매","운동시설"].some(u => 용도.includes(u));
const 단높이 = 계단강화용도 ? "18㎝" : "20㎝";
const 단너비 = 계단강화용도 ? "26㎝" : 공동주택 ? "24㎝" : "24㎝";
const 계단폭 = 계단강화용도 ? "150㎝" : 공동주택 ? "120㎝" : "120㎝";
items.push({ category:"다", 항목:"계단 치수", 법령:"건축법 시행령 제48조의2, 별표1의2",
    내용:`단높이 **${단높이} 이하**, 단너비 **${단너비} 이상**, 계단 유효너비 **${계단폭} 이상**`,
    해당여부: Y(`계단 치수 기준 — 단높이 ${단높이}↓ · 단너비 ${단너비}↑ · 유효너비 ${계단폭}↑`),
    설계기준:"세대 내 전용계단 완화: 단높이 20㎝↓, 단너비 20㎝↑, 너비 90㎝↑" });
```

### 5. 계단·복도 난간

- **적용 위치**: 건축물 내부
- **정확한 법령명**: 건축법 시행령
- **조문**: 제48조의3 (계단의 난간)
- **별표**: 없음
- **적용 대상**: 높이 1m 초과 계단, 높이 1.2m 이상 옥외통로·복도·발코니
- **기준 내용**:
  - 계단 너비 **3m 이상**: 중간 난간 추가 설치
  - 난간 높이: **1.0m 이상**
  - 복도·발코니 추락방지 난간: 높이 **1.2m 이상** 부분에 설치 (주택건설기준 제17조)
  - 난간 간살 간격: **10㎝ 이하** (아동 추락 방지, 주택건설기준 적용)
- **필요 입력값**: 층수, 용도, 계단 너비
- **판단 가능 여부**: 기준 표시 가능. 실제 설치 여부는 도면 필요
- **공식 URL**: https://www.law.go.kr/법령/건축법시행령/제48조의3
- **confidence**: confirmed (높이 1.0m), estimated (간살 10㎝ — 주택건설기준 적용 범위 확인 필요)
- **기존 시스템 매칭**: **누락**
- **조치**: **신설 필요**

**신설 코드 (목표):**
```typescript
items.push({ category:"다", 항목:"계단·난간", 법령:"건축법 시행령 제48조의3",
    내용:"계단 난간 높이 **1.0m 이상**, 너비 3m↑ 계단 중간난간 추가. 발코니·옥외복도: 높이 **1.2m 이상** 추락방지 난간",
    해당여부: Y("계단 난간 1.0m↑, 발코니·옥외복도 1.2m↑ 난간 설치"),
    설계기준:"난간 간살 간격 10㎝↓ (아동 추락 방지). 중간 난간: 계단 너비 3m↑ 시 추가" });
```

### 6. 접도 기준

- **적용 위치**: 도로
- **정확한 법령명**: 건축법
- **조문**: **제44조** (대지와 도로의 관계), 시행령 **제28조**
- **별표**: 없음
- **적용 대상**: 건축허가 대상 모든 건축물
- **기준 내용**:
  - 기본: 대지 → 너비 **4m 이상** 도로에 **2m 이상** 접도
  - 연면적 2,000㎡ 이상 (공장 3,000㎡): 너비 **6m 이상** 도로에 **4m 이상** 접도
  - 맹지 예외 인정: 건축허가권자 인정 시, 주변 공지 이용 시
- **필요 입력값**: 연면적, 인접도로 너비
- **판단 가능 여부**: 연면적 입력 시 필요 도로폭 표시 가능. 실제 접도 여부는 지적도·현장 확인 필요
- **공식 URL**: https://www.law.go.kr/법령/건축법/제44조
- **confidence**: confirmed
- **기존 시스템 매칭**: 두 항목 분산 — `"접도 요건 (맹지 여부)"`, `"접도(큰건물)"` — 내용 대부분 정확하나 도로폭 조건 구분 로직 확인 필요
- **조치**: 병합 또는 구조 정리 권장. 맹지 여부 + 도로폭 기준을 단계별로 표시

### 7. 가각전제 / 도로 모퉁이 건축선

- **적용 위치**: 도로
- **정확한 법령명**: 건축법, 지방자치단체 조례
- **조문**: 건축법 **제46조** 제1항 (건축선의 지정), 시행령 **제31조**
- **별표**: 없음
- **적용 대상**: 도로 교차각 120° 미만인 모퉁이 대지
- **기준 내용**:
  - 교차각 **90° 미만**: 가각부 **4m × 4m** 삼각형 내 건축 불가
  - 교차각 **90° 이상 ~ 120° 미만**: **3m × 3m** 삼각형 내 건축 불가
  - 교차각 **120° 이상**: 해당 없음
  - 단, 지자체 조례로 달리 정할 수 있음
- **필요 입력값**: 도로 교차각, 인접 도로 너비
- **판단 가능 여부**: 교차각 입력 없으면 판단 불가. 현재 ⚠️ 처리는 적절함
- **공식 URL**: https://www.law.go.kr/법령/건축법시행령/제31조
- **confidence**: confirmed (원칙), estimated (가각거리 수치 — 조례마다 상이)
- **기존 시스템 매칭**: `category:"자(도로)"`, `항목:"가각지 건축제한"` — **법령을 제46조로 기재하나 시행령 제31조가 정확**

### 8. 지하층 구조

- **적용 위치**: 건축물 내부
- **정확한 법령명**: 건축법, 건축법 시행령
- **조문**: 건축법 **제53조** + 건축법 시행령 **제64조**
- **별표**: 없음
- **적용 대상**: 거실을 두는 지하층이 있는 모든 건축물
- **기준 내용**:
  - 채광: 거실 바닥면적의 **1/10 이상** 채광창
  - 환기: 거실 바닥면적의 **1/20 이상** 환기창 또는 기계환기
  - 방습: 지하층 벽·바닥 방습층 처리
  - 지하층 바닥면적 합계 **1,000㎡ 이상**: 기계식 환기설비 의무
  - **비상탈출구**: 거실 바닥면적 **50㎡ 이상** 지하층에 비상탈출구 또는 2개소 이상 출구 확보 (현행 코드 누락)
- **필요 입력값**: 지하층 유무, 지하층 바닥면적, 거실 여부
- **판단 가능 여부**: 지하층 입력 시 판단 가능
- **공식 URL**: https://www.law.go.kr/법령/건축법시행령/제64조
- **confidence**: confirmed
- **기존 시스템 매칭**: `category:"파"`, `항목:"지하층의 구조"` — **비상탈출구(50㎡ 기준) 누락**
- **조치**: 비상탈출구 기준 내용 추가

**수정 후 (목표):**
```typescript
const 지하거실면적 = 지하층 > 0 && hasTotalArea ? 면 / (지하층 + 층) : 0; // 추정
items.push({ category:"파", 항목:"지하층의 구조", 법령:"건축법 제53조, 시행령 제64조",
    내용:"지하층 거실: 채광창 **1/10↑**, 환기창 **1/20↑** (1,000㎡↑ 기계환기). 바닥면적 **50㎡↑** 지하거실: 비상탈출구 또는 출구 2개소↑",
    해당여부: 지하층 > 0
      ? Y(`지하 ${지하층}층 — 채광·환기·방습 기준${면/지하층 >= 1000 ? " + 기계환기 의무" : ""}. 거실면적 50㎡↑ 시 비상탈출구 확인`)
      : N });
```

### 9. 장애인 편의시설

- **적용 위치**: 장애인
- **정확한 법령명**: 장애인·노인·임산부 등의 편의증진 보장에 관한 법률 (편의증진법)
- **조문**: **제7조** (편의시설의 설치), 시행령 **제3조~제4조**
- **별표**: 시행령 **별표2** (편의시설의 종류 및 설치기준), 시행규칙 **별표1**
- **적용 대상**:

  | 시설 유형 | 면적 기준 |
  |---------|---------|
  | 공동주택 | 세대수 무관 (모든 공동주택) |
  | 의료시설 | 500㎡ 이상 |
  | 근린생활시설 | 1종 500㎡ 이상, 2종 500㎡ 이상 |
  | 업무시설 | 500㎡ 이상 |
  | 판매시설 | 500㎡ 이상 |
  | 문화집회시설 | 500㎡ 이상 |
  | 교육연구시설 | 500㎡ 이상 |

- **주요 편의시설 기준**:
  - 접근로: 유효폭 **1.2m↑**, 경사도 **1/18 이하**
  - 주출입구: 유효폭 **0.9m↑**
  - 화장실 활동공간: **1.4m × 1.4m↑**
  - 장애인주차: 주차대수의 **2~4% (최소 1면)**
  - 점자블록: 주출입구~보도 연결
  - 장애인 전용 승강기: 6층↑ 건축물 (내부 규격 기준 있음)

- **필요 입력값**: 용도, 연면적, 주차대수
- **판단 가능 여부**: 가능 (단, 세부 시설별 기준은 별표2 상세 검토 필요)
- **공식 URL**: https://www.law.go.kr/법령/장애인ㆍ노인ㆍ임산부등의편의증진보장에관한법률시행령/별표2
- **confidence**: confirmed (대상시설 분류), estimated (세부 면적 기준 — 별표2 전체 미반영)
- **기존 시스템 매칭**: `category:"차"`, `항목:"장애인 편의시설"` — 면적 기준이 불완전 (근린 300㎡ 오기)
- **조치**: 별표2 기준으로 대상시설 목록 재정비. 근린생활시설 300㎡ → **500㎡**로 수정

### 10. 주요 소방시설 설치 기준

- **적용 위치**: 소방
- **정확한 법령명**: 소방시설 설치 및 관리에 관한 법률 (소방시설법)
- **조문**: 시행령 **제11조** (소방시설 설치)
- **별표**: 시행령 **별표4** (소방시설 설치 기준)
- **적용 대상**: 특정소방대상물 (용도·규모별 구분)

#### 현행 vs 정확한 기준 비교

| 소방시설 | 정확한 기준 | 현행 코드 | 상태 |
|---------|-----------|---------|------|
| 소화기 | 연면적 33㎡↑, 각 층 1개 | ✅ 정확 | OK |
| 옥내소화전 | 연면적 3,000㎡↑ **또는** 지하층·무창층 600㎡↑ | 지하층 600㎡ 기준 누락 | ⚠️ 보완 |
| 스프링클러 | 11층↑, 공동주택 6층↑, 노유자, 숙박 지하포함 600㎡↑ | 숙박 기준 불완전 | ⚠️ 보완 |
| **자동화재탐지설비** | 연면적 **400㎡↑** (근린·의료·업무·판매·숙박·문화집회 **300㎡↑**), 공동주택 **600㎡↑** | **누락** | ❌ 신설 |
| **비상방송설비** | 연면적 **3,500㎡↑** 또는 **11층↑** | **누락** | ❌ 신설 |
| **유도등** | 피난층 제외 모든 층 | **누락** | ❌ 신설 |
| **비상조명등** | 지하층·무창층·11층↑ 거실 | 누락 (별도 항목) | ❌ 신설 |

- **필요 입력값**: 용도, 연면적, 층수, 지하층 유무
- **판단 가능 여부**: 가능
- **공식 URL**: https://www.law.go.kr/법령/소방시설설치및관리에관한법률시행령/별표4
- **confidence**: confirmed (소화기·옥내소화전·스프링클러), estimated (스프링클러 세부 용도별 분기)

**신설 코드 — 자동화재탐지설비 (목표):**
```typescript
const 탐지강화용도 = ["근린생활","의료","업무","판매","숙박","문화집회"].some(u => 용도.includes(u));
const 자탐기준 = 탐지강화용도 ? 300 : 공동주택 ? 600 : 400;
const 자탐해당 = hasTotalArea && 면 >= 자탐기준;
if (자탐해당) items.push({ category:"사", 항목:"자동화재탐지설비", 법령:"소방시설법 시행령 별표4",
    내용:`연면적 **${자탐기준}㎡↑**: 자동화재탐지설비 의무`,
    해당여부: Y(`자동화재탐지설비 설치 (감지기·수신기·발신기)`),
    설계기준:"감지기 종류: 연기·열 복합형. 수신기 위치: 방재실 또는 주요 출입구" });
```
